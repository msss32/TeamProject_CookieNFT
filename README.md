## 📌 프로젝트 소개

<aside>
🍪 ERC20, ERC721 기반 쿠키런 카드 NFT 거래소

</aside>

- Solidity를 사용해 컨트랙트 구현
- Web3Js로 메타마스크와 컨트랙트 연결
- IPFS를 사용해 메타데이터와 이미지 저장
- 뽑기 애니메이션 구현

<br/>

## 📍개발 환경

- JavaScript
- Styled-Components
- React
- Redux
- Solidity
- IPFS(pinata)
